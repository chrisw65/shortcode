# OkLeaf Browser Extension - Store Listing Pack

## Short description (Chrome/Edge)
Shorten the current tab and copy the branded link instantly.

## Long description (Chrome/Edge)
OkLeaf Shortlink lets you create branded short links from any page without leaving your workflow.

Key features:
- Shorten the current tab in one click.
- Copy the short URL automatically.
- Uses your OkLeaf API key for secure access.
- Works with custom domains set in your OkLeaf account.

No browsing history is collected. The extension only sends the current tab URL to OkLeaf when you click the action.

## Firefox summary
Shorten the current tab and copy the short URL instantly.

## Permissions justification
- activeTab, tabs: read the current tab URL when you click the extension.
- clipboardWrite: copy the short URL to the clipboard.
- storage: save your API key locally in the browser.
- host_permissions <all_urls>: allow shortening any site.
