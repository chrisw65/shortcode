Authentication

Create an API key in the admin panel and pass it as a Bearer token.

curl -X GET https://okleaf.link/api/links \
  -H "Authorization: Bearer <YOUR_API_KEY>"

Create a short link

Send the long URL and optional title or custom code.

curl -X POST https://okleaf.link/api/links \
  -H "Authorization: Bearer <YOUR_API_KEY>" \
  -H "Content-Type: application/json" \
  -d '{"original_url":"https://example.com/launch","title":"Launch"}'

Analytics summary

Pull clicks and referrers for a specific link.

curl -X GET https://okleaf.link/api/analytics/links/<LINK_ID>/summary \
  -H "Authorization: Bearer <YOUR_API_KEY>"

Helpful endpoints

POST /api/links - create a link

GET /api/links - list links

POST /api/domains - add a domain

GET /api/analytics/domains/:id/summary - domain analytics

SDKs

Use the OkLeaf SDKs to wrap authentication and requests.

JavaScript / TypeScript

Install @okleaf/sdk and create links with typed clients.

npm install @okleaf/sdk

import { OkLeafClient } from '@okleaf/sdk';

const client = new OkLeafClient({ apiKey: 'YOUR_KEY' });
await client.links.create({ originalUrl: 'https://example.com' });

Webhooks

Push click events to your data warehouse or CRM in realtime.

Event types: link.created, link.deleted, click.recorded

Retries: 5 attempts over 10 minutes

Signature: HMAC SHA-256 in X-Okleaf-Signature

  "type": "click.recorded",

  "ip": "203.0.113.4",
